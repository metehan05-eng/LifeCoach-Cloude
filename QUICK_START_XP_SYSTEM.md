# ⭐ XP & 🔥 Alev Seviyesi Sistemi - Hızlı Başlangıç

## Sistem Nedir?

**XP (Experience Points)** ve **Alev Seviyesi** sistemi, kullanıcıların LifeCoach'ta verdikleri tüm çabaları ödüllendiren bir gamification sistemidir.

### Temel Kavramlar

| Kavram | Açıklama | Örnek |
|--------|----------|--------|
| **XP** | Her aktivitede kazanılan puan | Hedef tamamla: +5 XP |
| **Level** | 100 XP = 1 Level | 250 XP = Level 3 |
| **Alev 🔥** | Premium özellikler için gereken para birimi | Waffle: -10 🔥 |

---

## 🎯 XP Nasıl Kazanılır?

### Günlük Aktiviteler
```
✅ Hedef Tamamla (Günlük)        → +5 XP, +5 🔥
✅ Hedef Tamamla (Haftalık)      → +20 XP, +20 🔥
✅ Hedef Tamamla (Aylık)         → +50 XP, +50 🔥
✅ Hedef Tamamla (Yıllık)        → +50 XP, +20 🔥
```

### Alışkanlıklar
```
✅ Günlük Hedef Tutturma         → +5 XP, +5 🔥
   (Örn: "Çay 8 bardak" hedefi karşılanırsa)
```

### Odaklanma Seansları
```
✅ Odaklanma Seansı Bitir        → +10 XP, +5 🔥
   (25 dakikalık seans: +10 XP)
```

### Yansıma & Günlük
```
✅ Yansıma Yazma                 → +5 XP, +10 🔥
✅ Duygu Günlüğü Yazma           → +5 XP, +10 🔥
```

### Planlar
```
✅ Günlük Plan Tamamla           → +10 XP, +10 🔥
✅ Haftalık Plan Tamamla         → +15 XP, +15 🔥
✅ Aylık Plan Tamamla            → +25 XP, +25 🔥
✅ Proje Plan Tamamla            → +500 XP, +100 🔥 ⭐
```

### Görev Parçalama
```
✅ 7 Günlük Plan Bitir           → +7 XP, +3 🔥
✅ 14 Günlük Plan Bitir          → +10 XP, +7 🔥
✅ 30 Günlük Plan Bitir          → +15 XP, +12 🔥
✅ 90 Günlük Plan Bitir          → +40 XP, +20 🔥
```

## 🔥 Alev Seviyesi Nasıl Kullanılır?

### Waffle (AI Görüntü Oluşturma)
```
Maliyet: -10 🔥 per görüntü

Adımlar:
1. "Waffle" bölümüne git
2. Görsel açıklaması yaz (prompt)
3. "Görüntü Oluştur" butonuna bas
4. 10 alev harcanır
5. AI resim oluşturulur ✨
```

### Diğer Premium Özellikler (Gelecek)
- 🎨 Özel Tema Paketi (-20 🔥)
- 📊 İleri Analitikler (-15 🔥)
- 🤖 VIP AI Koçu (-50 🔥)

## 🎮 Stratejiler

### En Mükemmel Ödül Nasıl Alınır?
```
1. Yıllık Hedef Koy      → +50 XP, +20 🔥 (bir kez)
2. Proje Plan Yap        → +500 XP, +100 🔥 (büyük kazanç!)
3. 90 Günlük Plan        → +40 XP, +20 🔥
4. Günlük Alışkanlıklar  → +5 XP, +5 🔥 (tekrarlı, tutarlı)
```

### Level'i Hızlı Atlamak İçin
```
💡 TİP: Proje Planlar 500 XP verir! En iyi seçenek.
```

### Alev Biriktirme
```
1. Yansıma yazma       → +10 🔥
2. Günlük yazma        → +10 🔥
3. Haften planlamak    → +15 🔥
= Haftalık ~50+ 🔥 (5 Waffle)
```

## ❌ Hatalı Durum Yönetimi

### "Yeterli alev seviyesi yok"
```
Gerekli: 10 🔥
Mevcut: 5 🔥

Çözüm:
→ Yansıma yazma (+10 🔥)
→ Günlük yazma (+10 🔥)
→ Hafta planı tamamla (+15 🔥)
```

### "Token gerekli"
```
Giriş yapmamışsın.

Çözüm:
→ Login sayfasına git
→ Hesapla giriş yap
→ Tekrar dene
```

### Stats güncellenmiyor
```
Çözüm:
→ Sayfayı yenile (F5)
→ Hard refresh (Ctrl+Shift+R)
→ Cache temizle (Ayarlar → Clear Cache)
```

## 📞 Teknik Bilgiler

### API Endpoints (Geliştiriciler İçin)

```javascript
// Stats Alma
GET /api/user-stats
Response: { xp: 250, flameLevel: 75, level: 3, ... }

// Ödül Ekleme
POST /api/user-stats
Body: { rewardType: "goal_daily" }

// Alev Harcama
POST /api/user-stats
Body: { action: "consume", consumeType: "waffle_ai_image" }

// Waffle
POST /api/waffle
Body: { prompt: "açık plaj", style: "realistic" }
```

### Frontend Services

```javascript
// XP ve Alev Al
const stats = await StatsService.getStats();

// Ödül Ver
await StatsService.addReward('goal_daily');

// Alev Harça
try {
    await StatsService.consumeFlame('waffle_ai_image');
} catch (e) {
    console.error("Alev yok!", e);
}
```

## 💡 İpuçları

1. **Tutarlılık Önemli**: Günlük aktiviteler daha az XP verir ama tutarlı gelir.
2. **Büyük Projeler**: Proje planlar 500 XP ile en büyük kazançtır.
3. **Alev Optimizasyon**: Yansıma ve günlük yazmak alev için en iyi yol.
4. **Level Meşalesi**: Level atladığında tüm stats reset olmaz, sadece level artar.
