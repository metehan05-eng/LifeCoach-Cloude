📐 Responsive Breakpoints

  • Mobile: < 640px (Telefon)
  • Tablet: 640px - 1023px (Tablet)
  • Desktop: 1024px - 1279px (Masaüstü)
  • Desktop XL: ≥ 1280px (Geniş Ekran)

✨ Responsive Sistem Özellikleri

  • CSS Variables: --page-padding, --section-gap, --button-height vb.
  • Mobile-First: Mobil önce tasarım, sonra desktop optimize
  • Accessible: Focus states, high contrast, keyboard navigation
  • Performance: prefers-reduced-motion, optimized animations
  • Touch Support: Safe area, notch support, 48px touch targets
  • Orientation: Landscape/portrait optimizasyonu
  • Print Styles: Yazdırma için optimize edilmiş
  • Accessibility: Focus indicators, semantic HTML

📚 Kullanım Kılavuzu

HTML'de Responsive Sınıfları Nasıl Kullanılacağı:

<!-- Grid System --> <div class="responsive-grid">...</div> <!-- Responsive Container --> <div class="container-responsive">...</div> <!-- Flex Layout --> <div class="flex-responsive">...</div> <!-- Spacing --> <div class="p-responsive">...</div> <div class="gap-responsive">...</div> <!-- Responsive Show/Hide --> <div class="hide-mobile">...</div> <!-- Sadece tablet+ --> <div class="hide-tablet">...</div> <!-- Sadece mobile + desktop --> <div class="hide-desktop">...</div> <!-- Sadece mobile + tablet -->
